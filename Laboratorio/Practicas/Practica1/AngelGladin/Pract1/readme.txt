== Breve descripción de como se realizó la práctica. ==
La práctica se dividió en tres partes; implementar un 
programa que da las soluciones de un polinomio de 2do
grado, un programa que cifra un texto, y otro que hace
operaciones sobre números binarios.

== Forma de ejecución de su programa ==
Para ejecutar el programa se deberá abrir una terminal
en donde se encuentra el código (en la directorio /src)
y se ejecutará el interprete de Haskell:
    ghci
Y dentro de el intérprete poner
    :l <ruta_Archivo>
O si se quiere cargar un archivo al intérprete basta poner:
    ghci <ruta_Archivo>

== Conclusiones ==
La práctica sirvió como un buen acercamiento a la programación
funcional y el hecho de usar Haskell lo considero aún mejor.
La forma de manipular números binarios usando un tipo de dato
definido por nosotros, lo encontré algo díficil al principio.
